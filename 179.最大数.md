##### 179. 最大数
``` hml
给定一组非负整数 nums，重新排列每个数的顺序（每个数不可拆分）使之组成一个最大的整数。

注意：输出结果可能非常大，所以你需要返回一个字符串而不是整数。
```

**示例 1：**
``` 
输入：nums = [10,2]

输出："210"
```

**示例 2：**
```
输入：nums = [3,30,34,5,9]

输出："9534330"
```

**示例 3：**
```
输入：nums = [1]

输出："1"
```


**解答：**

``` javascript
/**
 * @param {number[]} nums
 * @return {string}
 */
 var largestNumber = function(nums) {
    nums.sort((a,b) => {
        if(a == b) {
            return 0;
        }
        a = a.toString(), b = b.toString();
        return (b + a) - (a + b);
    });
    let result = nums.join('');
    return result == 0 ? '0' : result;
};
```
